# Soltys.ChangeCase

Implementing https://www.npmjs.com/package/change-case in C#

## Installation 
```
  Install-Package Soltys.ChangeCase
```

## Usage
Add using:
```csharp
  using Soltys.ChangeCase
```

Sample usage:
```csharp
  string camelCase = "test string".CamelCase() // camelCase => testString
```
